    - Value within bounds returns the value (e.g., Clamp(5, 0, 10) == 5)
    - Value below min returns min (e.g., Clamp(-1, 0, 10) == 0)
    - Value above max returns max (e.g., Clamp(11, 0, 10) == 10)
    - Exact boundary: Clamp(0, 0, 10) == 0, Clamp(10, 0, 10) == 10
    - Numeric string coercion: Clamp("5", 0, 10) == 5
    - Non-numeric string returns nil: Clamp("abc", 0, 10) == nil
    - Negative range: Clamp(-5, -10, -1) == -5

    DMX.Util.ParseHexColor tests (at least 5 cases):
    - 6-char hex returns {r, g, b, a=1}: ParseHexColor("b88c03") produces r ~= 0.72, a == 1
    - 8-char hex includes alpha: ParseHexColor("b88c0380") produces a ~= 0.5
    - Strips leading #: ParseHexColor("#ff0000") produces r == 1, g == 0, b == 0
    - Invalid chars return nil: ParseHexColor("zzzzzz") == nil
    - Wrong length returns nil: ParseHexColor("fff") == nil
    - Empty string returns nil: ParseHexColor("") == nil

    DMX.Util.ParseOnOff tests (at least 6 cases):
    - Truthy tokens: ParseOnOff("on") == true, ParseOnOff("true") == true, ParseOnOff("1") == true, ParseOnOff("yes") == true
    - Falsy tokens: ParseOnOff("off") == false, ParseOnOff("false") == false, ParseOnOff("0") == false, ParseOnOff("no") == false
    - Case insensitive: ParseOnOff("ON") == true, ParseOnOff("True") == true
    - Unrecognized returns nil: ParseOnOff("maybe") == nil
    - Whitespace trimmed: ParseOnOff("  on  ") == true

    DMX.Util.Trim tests (at least 4 cases):
    - Strips leading and trailing whitespace: Trim("  hello  ") == "hello"
    - Nil input returns empty string: Trim(nil) == ""
    - Whitespace-only returns empty: Trim("   ") == ""
    - No whitespace passes through: Trim("hello") == "hello"

    Use `assert.are.equal` for exact comparisons, `assert.is_nil` for nil checks, `assert.is_true`/`assert.is_false` for booleans. For floating point comparisons in ParseHexColor, use `assert.is_near(expected, actual, tolerance)` with tolerance 0.01.
